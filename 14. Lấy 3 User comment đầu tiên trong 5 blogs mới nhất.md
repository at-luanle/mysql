### 14. Lấy 3 User comment đầu tiên trong 5 blogs mới nhất
```mysql
use train_mysql;
  select user.*
  from user
  join
  (
	select comment.user_id, created_at
	from comment
	where  target_table = 'blog'
    order by comment.created_at desc
    limit 5
  )  as com
on com.user_id = user.id
limit 3



```
