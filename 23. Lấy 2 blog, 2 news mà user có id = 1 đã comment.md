### 23. Lấy 2 blog, 2 news mà user có id = 1 đã comment
```mysql
(	select  id, title  
	from blog where id in
											(select target_id
											from comment
											where user_id = 1 and target_table = 'blog' )
	limit 2)
union
(	select  id, title
	from news where id in
											(select target_id
											from comment
											where user_id = 1 and target_table = 'news' )  
	limit 2) 
```
